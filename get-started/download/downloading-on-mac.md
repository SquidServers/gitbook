# Downloading on Mac

## .zip steps

1. Download the ZIP file and unzip it (double-click).
2. Drag "SquidServers.app" to the "Applications" folder.
3. Open Terminal (press `Command + Space` and search for `Terminal`).
4. Run the command below if macOS blocks the app on first launch:

```sh
xattr -cr /Applications/SquidServers.app
```

## .dmg steps

1. Download the DMG file and open it.
2. Drag "SquidServers.app" to the "Applications" folder.
3. Optional: Eject the DMG.
4. Open Terminal (press `Command + Space` and search for `Terminal`).
5. Run the command below if macOS blocks the app on first launch:

```sh
xattr -cr /Applications/SquidServers.app
```
